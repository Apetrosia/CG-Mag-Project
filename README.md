# Атмосферная обсерватория (Fog + Skybox + Day/Night Cycle)

__Что делает:__ Вид на горы с небом, туманом и циклической сменой дня и ночи.

__Техника:__ Процедурное небо (градиент + солнце как спрайт), экспоненциальный туман по высоте, смешивание текстур земли.

__Для мобилок:__ Показать энергоэффективность — no real-time lights, только шейдеры.

__Сложность: ★★★__
